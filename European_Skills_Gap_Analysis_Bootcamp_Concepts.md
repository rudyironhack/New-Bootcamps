For each identified skills gap:
1. **Current coverage** - Does existing portfolio address this?
2. **Overlap assessment** - Would new bootcamp cannibalize existing programs?
3. **Content sufficiency** - Is there 9 weeks (400h) of original content?
4. **Standalone viability** - Can this be distinct or should it be integrated?

- Overlap percentages are subjective estimates without formal quantitative methodology
- Salary projections are unsourced estimates based on market research
- No primary research conducted (employer interviews, student demand signals)
- Missing financial analysis (break-even, ROI, development costs)

### **CONCEPT 1: Cloud & AI Security Engineering**

**What is Cloud & AI Security Engineering?**
Cloud & AI Security Engineering is the discipline of protecting cloud infrastructure (AWS, Azure, GCP) and artificial intelligence systems from cyber attacks. It includes identity and access management (IAM), container and Kubernetes security, cloud data protection, and compliance with regulations like GDPR and SOC2. The AI Security component addresses emerging threats specific to AI systems: adversarial attacks (fooling models with manipulated inputs), model poisoning (corrupting training data), prompt injection (manipulating LLMs), and privacy attacks (extracting sensitive data from models).

**Strategic Rationale:**
Merge Cloud Security (mature discipline) + AI Security (emerging field) into unified "Security Engineer for Cloud+AI Era" bootcamp. Cloud Security alone justifies 6-7 weeks, AI Security adds 2-3 weeks of differentiation.

**Market Evidence:**
- 4.76M cybersecurity workforce gap globally (ISC2 2024 figure; ISC2 stopped tracking headcount gaps in 2025). *EU-specific gap: 274,000 (ISC2 2023)* ([ISC2](https://www.isc2.org/Insights/2025/10/bridging-EU-cybersecurity-skills-gap-ECSF-and-certification-mapping))
- 89% of security teams report skills gaps ([ISC2](https://www.isc2.org/Insights/2025/10/bridging-EU-cybersecurity-skills-gap-ECSF-and-certification-mapping))
- Zero bootcamps combining cloud security + AI security in Europe

**Why Minimal Overlap with Current Portfolio:**
- Ironhack's Cybersecurity bootcamp covers **general security** (network, SIEM, pentesting, malware)
- This bootcamp is **cloud-specific** (AWS/Azure security, IAM, K8s security, compliance)
- *Note: Overlap assessment may be understated - likely 30-35% overlap with existing Cybersecurity and Cloud Engineering bootcamps*

**Core Topics:**
- Cloud fundamentals & architecture (AWS, Azure, GCP)
- Cloud IAM, identity federation, SSO
- Network security (VPC, firewalls, WAF, zero trust)
- Encryption, key management, secrets management
- Compliance frameworks (GDPR, SOC2, ISO27001)
- Container & Kubernetes security
- **AI Security** (adversarial ML, model poisoning, prompt injection, AI governance)
- Cloud incident response & automation

**Target Roles:**
Cloud Security Engineer, Cloud Security Architect, AI Security Specialist, DevSecOps Engineer

**Salary Outcomes (Europe):**
€45K-55K entry → €65K-80K mid → €90K-120K+ senior

**Vendor Certifications:**
AWS Certified Security - Specialty, Azure Security Engineer Associate, CCSP prep

**Hardware:** ✅ Standard laptop - All labs in cloud environments

### **CONCEPT 2: Computer Vision Engineering**

**What is Computer Vision Engineering?**
Computer Vision Engineering is the AI specialization that teaches machines to "see" and interpret images and video. It includes techniques for image classification (identifying what's in an image), object detection (locating where objects are), segmentation (separating objects from background), video tracking (following objects in motion), and 3D vision (reconstructing depth and space). Applications span autonomous vehicles, medical imaging, industrial quality control, augmented reality, retail analytics, and intelligent surveillance systems.

**Strategic Rationale:**
Deep specialization in AI-powered image/video analysis. AI Engineering touches CV superficially (~1 project), this is 9-week deep dive into visual AI.

**Market Evidence:**
- Computer Vision is a key skill in high-growth AI roles: AI/ML Researcher (#5 fastest-growing role) and AI Consultant (#2) both list Computer Vision among their most common skills ([LinkedIn Jobs on the Rise 2026](https://www.linkedin.com/pulse/linkedin-jobs-rise-2026-25-fastest-growing-roles-us-linkedin-news-dlb1c/))
- While "Computer Vision Engineer" is not a separate top-ranked role, computer vision expertise is in high demand as a specialization within AI/ML roles
- Average salary €70-95K in Europe (€110K+ is senior-level only, not average)
- Only Datascientest offers dedicated Computer Vision bootcamp in Europe ("Deep Learning for computer Vision") - minimal competition
- *Concerns: Niche market, inflated salary data in initial research, instructor scarcity*
- Applications: autonomous vehicles, medical imaging, AR/VR, industrial inspection, retail analytics

**Why Minimal Overlap:**
- AI Engineering: Broad (ML + DL + NLP + CV + LLM), CV is ~10% of curriculum
- CV Bootcamp: Specialist (100% visual AI), 9 weeks depth
- Different job titles: "AI Engineer" (generalist) vs. "Computer Vision Engineer" (specialist)

**Core Topics:**
- Classical CV (OpenCV, filters, feature detection)
- Deep learning for vision (CNNs, Vision Transformers)
- Object detection (YOLO, Faster R-CNN)
- Segmentation (U-Net, Mask R-CNN, SAM)
- Video analytics and tracking
- 3D vision and depth estimation
- Domain applications (medical imaging, autonomous systems)
- Production deployment (TensorRT, ONNX, edge inference)

**Target Roles:**
Computer Vision Engineer, CV Research Engineer, Autonomous Systems Engineer, Medical Imaging AI Specialist

**Salary Outcomes (Europe):**
€50K-65K entry → €70K-90K mid → €95K-110K+ senior (premium over general AI, but initial €110K+ average claim was inflated)

**Vendor Certifications:**
NVIDIA Deep Learning Institute - CV certification, AWS ML Specialty (CV focus)

**Hardware:** ✅ Standard laptop - GPU access via cloud (AWS, GCP, Colab)

### **CONCEPT 3: Green Tech & AI for Sustainability**

**What is Green Tech & AI for Sustainability?**
Green Tech & AI for Sustainability applies data science and artificial intelligence to environmental and climate challenges. It includes: forecasting renewable energy production (solar, wind), optimizing smart grids and energy demand management, tracking CO2 emissions and corporate carbon footprint, ESG (Environmental, Social, Governance) reporting systems required by EU CSRD regulations, climate data analysis and environmental monitoring, and predictive maintenance for green infrastructure. This is a "purpose-driven" bootcamp that attracts professionals motivated by sustainability.

**Strategic Rationale:**
Complete white space - zero bootcamps addressing sustainability tech in Europe. Purpose-driven positioning attracts net-new demographic (climate-motivated professionals).

**Market Evidence:**
- Spain: 14.9% employment growth in electricity/gas sector ([Spain Education Monitor](https://op.europa.eu/webpub/eac/education-and-training-monitor/en/country-reports/spain.html))
- EU Green Deal: €1 trillion investment 2024-2030 ([European Commission Green Deal](https://commission.europa.eu/strategy-and-policy/priorities-2019-2024/european-green-deal_en)) - *Note: Total includes public and private investment*
- ESG reporting requirements (EU CSRD) mandate sustainability data specialists
- **Competitive landscape:** 
  - **Global programs:** Columbia LEAP Climate Data Science bootcamp, Stanford AI for Climate Change bootcamp, MIT AI for Energy Solutions, SkillX AI Renewable Energy Analytics - but these are primarily academic/university programs or short-format courses (2-5 days, 4-8 weeks) vs. full 9-week bootcamp format
  - **European programs:** 
    - Talent Garden (Italy): Business Sustainability Master (part-time) - business-focused, covers ESG ratings, sustainable finance, digital sustainability
    - Bocconi/Politecnico di Milano (Italy): Transformative Sustainability MSc (2-year university program)
    - EIT Digital: Data Science with Specialization in Sustainability Master's (in development)
    - AI4GreenDeal: EU-funded double-degree Master's combining energy systems engineering, AI, and data science
  - **Gap:** No European commercial bootcamp in traditional 9-week format combining AI/data science with green tech/sustainability
- *Opportunity: First European commercial bootcamp in this space, but limited precedent suggests need for primary market validation*

**Why Zero Overlap:**
- Domain-specific: Energy systems, climate tech, sustainability analytics
- Ironhack has no climate/energy/sustainability content in any bootcamp

**Core Topics:**
- Data science fundamentals + energy systems basics *(covered in: AI4GreenDeal, EIT Digital, MIT, Columbia programs)*
- AI for renewable energy forecasting (solar, wind) *(covered in: SkillX AI, MIT, Renewable Energy Institute programs)*
- Smart grid analytics and demand prediction *(covered in: SkillX AI, some energy programs)*
- Carbon footprint tracking and ESG reporting systems *(covered in: Talent Garden, Bocconi programs - but more business-focused)*
- Climate data analysis and environmental monitoring *(covered in: Columbia, Stanford programs)*

### Strategic Considerations

**Why Healthcare is Separated:**

1. **Different Market Segment**: Healthcare education typically involves regulated credentials (nursing degrees, medical certifications) rather than bootcamp-style intensive training

2. **Regulatory Complexity**: Healthcare professions require licensing and accreditation that varies by country, making pan-European delivery more complex than tech bootcamps

3. **Different Competitive Landscape**: Healthcare training is dominated by universities, vocational schools, and specialized healthcare academies rather than tech bootcamp competitors

4. **Brand Positioning**: Ironhack's brand is built around technology and innovation; healthcare would represent significant brand extension

**Potential Adjacent Opportunities:**

If Ironhack wished to explore healthcare-adjacent offerings without full sector pivot:

1. **Health Tech Engineering** - Software development for healthcare (EHR systems, telemedicine platforms, medical device software)
2. **Healthcare Data Analytics** - Apply existing Data Analytics bootcamp to healthcare domain (patient outcomes analysis, clinical trial data, health economics)
3. **AI for Healthcare** - Specialized AI Engineering track focused on medical imaging, diagnostics, drug discovery

These would leverage existing tech bootcamp expertise while serving healthcare sector demand.

### Recommendation

Healthcare/care economy represents significant job growth but falls outside Ironhack's core competency and brand positioning. Monitor for future strategic review if considering broader education market expansion beyond tech.
